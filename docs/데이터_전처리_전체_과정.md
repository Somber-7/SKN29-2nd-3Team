| 항목 | 내용 |
|------|------|
| 원본 출처 | 국토교통부 아파트 실거래가 공개 데이터 |


#### 1-1. 시군구 컬럼 추가

지역코드(행정구역 5자리 코드)를 기반으로 시군구 명칭 컬럼을 새로 생성했습니다.

- 매핑 기준: 지역코드 → 시군구 행정구역명 (전국 249개 시군구)
- 결측: 377,521건 (일부 지역코드 미매핑 — 이후 Step 3에서 보완)

```
원본:  지역코드만 존재 (예: 45111)
변환:  시군구 컬럼 추가 (예: 전라북도 전주시 완산구)
```

#### 1-2. 지역코드 타입 변환

소수점이 포함된 float 타입을 정수형으로 변환했습니다.

```
원본:  float64  (예: 31110.0)
변환:  int64    (예: 31110)
```

#### 1-3. 거래일 형식 통일

날짜 형식을 ISO 8601 표준으로 통일했습니다.

```
원본:  M/D/YYYY H:MM  (예: 5/30/2020 0:00)
변환:  YYYY-MM-DD     (예: 2020-05-30)
```

#### 1-4. 데이터 정렬

지역코드 오름차순으로 전체 데이터를 재정렬했습니다.

```
원본:  지역코드 비정렬 (울산 31110 시작)
변환:  지역코드 오름차순 정렬 (서울 11110 시작)
```

#### 1-5. 컬럼 순서 재정렬

분석 편의에 맞게 컬럼 순서를 재배치했습니다.

```
원본:  지역코드 / 법정동 / 거래일 / 아파트 / 지번 / 전용면적 / 층 / 건축년도 / 거래금액
변환:  지역코드 / 시군구 / 법정동 / 지번 / 아파트 / 거래일 / 건축년도 / 층 / 전용면적 / 거래금액
```
