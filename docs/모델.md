# 모델 정리

아파트 실거래가 데이터 프로젝트에서 지금까지 작성한 모델 파일 정리입니다.  
전체 구조는 `models/base.py`의 `BaseModel`, `BaseClusterModel`을 상속하는 방식으로 통일하는 것을 기준으로 합니다.

---

## 1. 공통 베이스 클래스

### 파일 위치

```text
models/base.py
```

### 역할

모든 모델이 동일한 인터페이스를 가지도록 하는 공통 추상 클래스입니다.

### 주요 클래스

```python
BaseModel
BaseClusterModel
```

### BaseModel 필수 메서드

```python
fit(X_train, y_train, **kwargs)
predict(X)
evaluate(X_test, y_test)
```

### BaseClusterModel 필수 메서드

```python
fit(X, **kwargs)
predict(X)
evaluate(X)
```

### 사용 목적

- 회귀 모델 공통 인터페이스 통일
- 분류 모델 공통 인터페이스 통일
- 군집화 모델 공통 인터페이스 통일
- Streamlit 화면에서 모델을 일관되게 호출하기 위함

- 기존 `eda.ipynb` 모델링 로직을 클래스화

- 원-핫 인코딩된 `지역코드`를 다시 합칠 필요 없이 원본 피처 기준 permutation importance 계산 가능

## 6. 이상 거래 탐지 모델

### 파일 위치

```text
models/anomaly/anomaly_transaction_model.py
```

### 클래스명

```python
AnomalyTransactionModel
```

### 목적

아파트 실거래 데이터에서 전체 패턴과 다른 **모델 기준 특이 거래**를 탐지하는 모델입니다.  
거래금액, 평당가, 위치, 건물연식, 주변 인프라, 세대수 등을 함께 사용하여 일반적인 거래 패턴에서 벗어난 데이터를 찾습니다.

> 주의: 이 모델의 결과는 실제 불법 거래나 허위 거래를 의미하지 않습니다.  
> 발표나 서비스 화면에서는 반드시 **“모델 기준 특이 거래”**, **“일반 패턴과 다른 거래”** 정도로 표현하는 것이 좋습니다.

## 7. 가격 프리미엄 분석기

### 파일 위치

```text
models/analysis/price_premium_analyzer.py
```

### 클래스명

```python
PricePremiumAnalyzer
```

### 목적

거래금액 예측 모델의 예측값과 실제 거래금액의 차이를 이용해 **브랜드, 역세권, 학세권, 대단지 프리미엄**을 분석하는 도구입니다.

이 클래스는 독립적인 예측 모델이라기보다는, 기존 회귀 모델을 활용한 **모델 기반 분석기**에 가깝습니다.

### 분석 흐름

```text
1. LightGBMPriceModel 같은 거래금액 예측 모델 학습
2. 실제 거래금액과 예측 거래금액 비교
3. 프리미엄금액 = 실제거래금액 - 예측거래금액
4. 프리미엄률 = 프리미엄금액 / 예측거래금액
5. 브랜드/역세권/학세권/대단지별 평균 프리미엄 분석
```

### 필요 모델

```python
LightGBMPriceModel
XGBoostPriceModel
ApartmentPriceModel
```

거래금액을 예측할 수 있는 회귀 모델이면 사용할 수 있습니다.

### 주요 생성 컬럼

```text
예측거래금액
프리미엄금액
프리미엄률
절대오차
오차율
프리미엄등급
역세권여부
학세권여부
대단지여부
브랜드구분
```

### 파생 변수 기준

```text
역세권여부: 인근역수 >= 1 이면 역세권
학세권여부: 인근학교수 >= 1 이면 학세권
대단지여부: 세대수 >= 1000 이면 대단지
브랜드구분: 브랜드여부 == 1 이면 브랜드, 아니면 비브랜드
```

### 주요 메서드

```python
analyze(df)
evaluate_price_model(premium_df)
summarize_by_group(premium_df, group_col)
compare_groups(premium_df, group_cols)
top_premium_transactions(premium_df, n=30)
top_discount_transactions(premium_df, n=30)
```

## 11. 발표용 핵심 메시지

이번 프로젝트의 모델 구성은 다음과 같이 설명할 수 있습니다.

```text
본 프로젝트는 아파트 실거래가 데이터를 기반으로
거래금액 예측, 브랜드 등급 분류, 입지 유형 군집화를 수행했습니다.

회귀 모델에서는 Linear Regression, RandomForest, LightGBM, XGBoost를 비교하여
대용량 tabular 데이터에서 어떤 모델이 가장 안정적인 성능을 보이는지 확인했습니다.

분류 모델에서는 브랜드명을 직접 입력하지 않고,
가격, 입지, 주변 인프라, 단지 규모만으로 브랜드 등급을 예측하도록 설계했습니다.

군집화 모델에서는 위도, 경도, 평당가, 역/학교 접근성, 세대수 등을 활용하여
아파트를 유사한 입지 유형으로 묶고 지도 기반으로 시각화할 수 있도록 구성했습니다.

이상 거래 탐지 모델에서는 IsolationForest를 사용하여 일반적인 거래 패턴과 다른 거래를 찾고,
가격 프리미엄 분석기에서는 예측 가격 대비 실제 거래가격의 차이를 기반으로
브랜드, 역세권, 학세권, 대단지 프리미엄을 비교했습니다.
```
