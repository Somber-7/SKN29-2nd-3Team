## utils/db.py — DB 연결

`conf/.env`의 접속 정보를 읽어 MySQL에 연결합니다.

### 주요 함수

| 함수 | 설명 |
|------|------|
| `get_connection()` | mysql.connector 연결 객체 반환 |
| `db_cursor(dictionary=True)` | context manager — 자동 commit/rollback |
| `fetch_all(query, params)` | SELECT → `list[dict]` 반환 |
| `fetch_one(query, params)` | SELECT → `dict \| None` 반환 |
| `execute(query, params)` | INSERT/UPDATE/DELETE → 영향 행 수 반환 |

### 사용 예시

```python
from utils.db import fetch_all, execute

rows = fetch_all("SELECT * FROM users WHERE age > %s", (20,))
execute("INSERT INTO logs (msg) VALUES (%s)", ("test",))
```

## utils/preprocessor.py — 전처리

| 함수 | 설명 |
|------|------|
| `get_scaler(method)` | `'standard'` \| `'minmax'` Scaler 반환 |
| `scale_features(X_train, X_test, method)` | 학습/테스트 스케일링 |
| `label_encode(series)` | Label 인코딩 → `(encoded, encoder)` |
| `onehot_encode(df, columns)` | One-Hot 인코딩 |
| `impute(df, strategy)` | 결측치 처리 (`mean` / `median` / `most_frequent`) |
| `build_pipeline(model, num_cols, cat_cols, scaler)` | sklearn Pipeline 생성 |
| `remove_outliers_iqr(df, columns, factor)` | IQR 기반 이상치 제거 |
| `log_transform(df, columns)` | `log1p` 변환 |
| `parse_price(df, col)` | 거래금액 str → int 변환 (쉼표 제거) |
| `parse_date(df, col)` | 거래일 → 거래년도 / 거래월 / 거래분기 파생 피처 추가 |
| `fix_floor(df, col)` | 층 -1 등 음수값 → NaN 처리 |
| `map_brand_grade(df, col)` | 브랜드 → 4등급(프리미엄/일반/공공/기타) 변환, `brand_grade` 컬럼 추가 |

### Apart Deal 전처리 순서 예시

```python
from utils.preprocessor import parse_price, parse_date, fix_floor, map_brand_grade, impute

df = parse_price(df)          # 거래금액 str → int
df = parse_date(df)           # 거래일 → 연/월/분기 파생 피처
df = fix_floor(df)            # 층 음수값 → NaN
df = map_brand_grade(df)      # 브랜드 → brand_grade 등급 컬럼 추가
df = impute(df, strategy="median")  # 남은 NaN 처리

# 분류 모델: brand_grade를 타깃으로, 브랜드/브랜드여부 컬럼 제거
# 회귀 모델: 거래금액을 타깃으로, brand_grade를 피처로 활용
```

## models/base.py — 모델 베이스 클래스

모든 지도학습 모델은 `BaseModel`을, 군집화 모델은 `BaseClusterModel`을 상속합니다.

### BaseModel (지도학습)

```python
from models.base import BaseModel

class MyModel(BaseModel):
    def __init__(self):
        super().__init__(name="MyModel")

    def fit(self, X_train, y_train, **kwargs) -> "MyModel":
        self._model = ...          # sklearn 모델 등 실제 모델 저장
        self._model.fit(X_train, y_train)
        self._is_trained = True
        return self

    def predict(self, X) -> np.ndarray:
        self._check_trained()
        return self._model.predict(X)

    def evaluate(self, X_test, y_test) -> dict:
        self._check_trained()
        y_pred = self.predict(X_test)
        return regression_metrics(y_test, y_pred)   # 또는 classification_metrics
```

### BaseClusterModel (군집화)

```python
from models.base import BaseClusterModel

class MyCluster(BaseClusterModel):
    def fit(self, X, **kwargs):
        self._model = ...
        self._model.fit(X)
        self._labels = self._model.labels_
        return self

    def predict(self, X):
        return self._model.predict(X)

    def evaluate(self, X):
        return clustering_metrics(X, self._labels)
```

### 제공 공통 메서드

| 메서드 | 설명 |
|--------|------|
| `get_params()` | 하이퍼파라미터 딕셔너리 반환 |
| `get_feature_importance()` | 피처 중요도 배열 반환 (지원 모델만) |
| `is_trained` (property) | 학습 완료 여부 bool |
| `_check_trained()` | 미학습 시 RuntimeError 발생 |

## 모델 구현 시 체크리스트

데이터 준비 완료 후, 각 영역별로 아래 순서로 작업합니다.

```
1. models/<영역>/<모델명>.py  — BaseModel 상속 구현
2. app/pages/<영역>.py       — Streamlit 페이지 작성
```

### 영역별 구현 대상

| 영역 | 파일 위치 | 구현 모델 |
|------|-----------|----------|
| 회귀 | `models/regression/` | LinearRegression, RandomForest, LightGBM |
| 분류 | `models/classification/` | SVM, DecisionTree, Ensemble |
| 군집화 | `models/clustering/` | KMeans, DBSCAN, Agglomerative |
| 차원 축소 | `models/dimensionality_reduction/` | PCA |
| 신경망 | `models/neural_network/` | MLP(sklearn), DNN(PyTorch) |
