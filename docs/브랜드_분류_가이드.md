## 다중 분류 등급 설계안

브랜드 프리미엄 외부 데이터 확보 후 아래 기준으로 등급을 확정할 예정입니다.
현재는 도메인 지식 기반 임시 분류안입니다.

### 4등급 분류안 (권장)

| 등급 | 포함 브랜드 | 비고 |
|------|------------|------|
| **프리미엄** | 래미안, 자이, 힐스테이트, 아이파크, 푸르지오, e편한세상, 더샵, 롯데캐슬, 아크로, SK뷰 | 상위 10개 브랜드 |
| **일반브랜드** | 위브, 하늘채, 베르디움, 린, 해링턴, 스위첸, 데시앙, 노블랜드, 스타힐스, 파밀리에 등 | 중소형 브랜드 |
| **공공(LH)** | LH(주공) | 민간 브랜드와 분리 |
| **기타** | 기타 및 비브랜드 소형 건설사 | |

### 3등급 분류안 (단순화)

| 등급 | 포함 |
|------|------|
| **프리미엄** | 상위 10개 브랜드 |
| **공공/일반** | LH + 중소형 브랜드 |
| **기타** | 비브랜드 |

---

## 전처리 시 처리 순서

```
1. 구/신 브랜드 병합 여부 결정
2. LH(주공) 처리 방향 결정 (별도 등급 or 제외)
3. 브랜드 프리미엄 외부 데이터 확보
4. 프리미엄 기준으로 등급 컬럼 생성 (brand_grade)
5. 기존 브랜드, 브랜드여부 컬럼 제거 (타깃과 피처 분리)
```
