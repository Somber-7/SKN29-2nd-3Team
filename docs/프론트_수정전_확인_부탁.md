구현 목표 디자인 분석 (REB 스타일)
요소	특징
좌측 사이드바	텍스트 메뉴 목록, 활성 항목 파란색 하이라이트 + 좌측 인디케이터
상단 헤더	페이지 제목(좌) + "→ 바로가기" 버튼(우) 양쪽 정렬
섹션 구분	컬러 원형 배지 아이콘 + 섹션명 + + 확장 버튼
콘텐츠 카드	흰 배경 + 둥근 모서리 + 그림자 + 아이콘 + "자세히보기 >"
수정 단계
Step 1 — CSS 사이드바 스타일 (가장 먼저)
현재 st.sidebar.radio는 기본 라디오 버튼 모양입니다.
style.css에 Streamlit 내부 클래스를 덮어써서 REB처럼 만들어야 합니다.

핵심 CSS 타깃:


[data-testid="stSidebarNav"] 항목들
→ 기본 라디오 버튼 숨기기
→ 전체 너비 클릭 영역으로 변환
→ hover 시 배경색 변경
→ 활성 항목: 좌측 4px 파란 border + 파란 텍스트 + 연파란 배경
추가로 사이드바 최상단에 로고/브랜드 영역 삽입:


st.sidebar.markdown('<div class="sidebar-logo">🏢 부동산 분석 플랫폼</div>', unsafe_allow_html=True)
Step 2 — 페이지 헤더 컴포넌트화
모든 페이지 상단을 아래 구조로 통일합니다:


[페이지 제목 (좌)]          [→ 바로가기 (우)]
─────────────────────────────────────────────
Streamlit에서 좌우 정렬은 st.columns로는 완벽하지 않아서, HTML flex 방식으로 처리:


def page_header(title, link_label="→ 바로가기"):
    st.markdown(f"""
    <div class="page-header">
        <h1>{title}</h1>
        <button class="header-btn">{link_label}</button>
    </div>
    <hr class="header-divider">
    """, unsafe_allow_html=True)
이 함수를 utils/ui.py에 분리해두면 모든 페이지에서 재사용 가능합니다.

Step 3 — 섹션 배지 + 카드 그리드 (개요 홈)
REB의 "월간동향", "주간아파트동향" 같은 섹션 헤더:


🔵 섹션명  ─────────────────── +
그 아래 카드 4개 가로 나열:


[아이콘]  [아이콘]  [아이콘]  [아이콘]
 제목      제목      제목      제목
자세히 >  자세히 >  자세히 >  자세히 >
구현 방식:

섹션 배지: CSS border-radius: 50% + 배경색 원 안에 아이콘
카드: st.markdown으로 HTML 카드 컴포넌트, st.columns(4)로 배치
아이콘: 이미지 없이 구현하려면 SVG inline 또는 CSS 폰트 아이콘 (Material Icons CDN 추가)
개요 홈의 4개 통계 카드는 REB 카드 스타일 적용:


┌─────────────────┐
│  5,000,000      │  ← 큰 숫자
│  총 거래 건수    │  ← 설명 텍스트
│  2015~2023      │  ← 보조 정보 (작게)
└─────────────────┘
Step 4 — 페이지별 카드 스타일 통일
가격 추이 분석:

지역 선택기 → REB처럼 탭 또는 버튼형 필터로 교체 (st.radio + CSS로 버튼처럼 보이게)
그래프 영역은 카드 박스 안에 넣기 (<div class="chart-card">)
입지 분석:

상관계수 카드 → REB 카드 스타일 통일
상단 3개(기준금리/인근역수/인근학교수) + 중간 2개(세대수/건축연식) + 하단 2개(역세권/학군지) 순서 유지
상관계수 숫자는 크게, 요인명은 작게 — 숫자 강조 방식
Step 5 — 멀티페이지 분리 + CSS 전역 적용
파일 구조 확정:


app/
├── Home.py                  ← set_page_config + load_css + 개요
├── pages/
│   ├── 1_가격추이분석.py
│   ├── 2_입지분석.py
│   ├── 3_지도시각화.py
│   └── utils/
│       └── ui.py            ← page_header(), section_badge(), stat_card() 공통 함수
assets/
└── css/
    └── style.css            ← 사이드바 + 카드 + 헤더 + 버튼 전체 스타일
load_css()를 각 페이지 최상단에서 호출하면 전체 스타일 통일됩니다.

작업 순서 요약
순서	파일	작업

3	Home.py	개요 페이지 카드 4개 + 차트 영역
4	pages/1_가격추이분석.py	필터 + 그래프
